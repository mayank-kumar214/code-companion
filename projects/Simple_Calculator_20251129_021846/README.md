# Simple Calculator

## Description
A lightweight, responsive web-based calculator that performs basic arithmetic operations. Built with pure HTML, CSS, and JavaScript, it works offline in any modern browser.

## Tech Stack
- **HTML5** – Semantic markup
- **CSS3** – Flexbox/Grid for responsive layout
- **JavaScript (ES6+)** – Calculator logic and UI interaction

## Features
- **Arithmetic operations** – addition, subtraction, multiplication, division
- **Clear (C) button** – reset the display and internal state
- **Responsive layout** – adapts to desktop, tablet, and mobile screens
- **Button interaction** – visual feedback on hover/press
- **Display** – shows the current input and results in real time

## Setup Instructions
1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/simple-calculator.git
   cd simple-calculator
   ```
2. **Open the application**
   - Locate the `index.html` file in the project root.
   - Double‑click it or open it directly in your web browser (Chrome, Firefox, Edge, Safari, etc.).
   - No additional server or build steps are required.

## Usage
- Click the numeric buttons (`0‑9`) to build a number.
- Use the operator buttons (`+`, `-`, `×`, `÷`) to choose an operation.
- Press `=` to evaluate the expression; the result appears in the display.
- Press `C` to clear the current entry and start a new calculation.
- The calculator also supports chaining operations (e.g., `5 + 3 × 2`).
