# SimpleCalculator

A lightweight web-based calculator that performs basic arithmetic operations. It provides a clean, responsive interface built with HTML, CSS, and JavaScript, allowing users to calculate quickly on both desktop and mobile devices.

---

## Tech Stack
- **HTML** – Structure of the calculator UI
- **CSS** – Styling and responsive layout
- **JavaScript** – Calculator logic and interactivity

---

## Features
- Basic arithmetic: addition, subtraction, multiplication, division
- Clear entry (Esc) and delete last digit (Backspace)
- Keyboard support for numbers, operators (`+`, `-`, `*`, `/`), `Enter` for equals, `Esc` for clear, and `Backspace` for delete
- Responsive design works on desktop browsers and mobile devices
- Simple, modular code: HTML for markup, CSS for styling, `script.js` for functionality

---

## Setup Instructions
1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd <repository-folder>
   ```
2. **Open the application**
   - Locate `index.html` in the project root.
   - Open `index.html` in any modern web browser (no build step or server required).

---

## Usage
- **Click Buttons**: Use the on‑screen buttons to input numbers and operations.
- **Keyboard Shortcuts**:
  - Numbers `0–9` – Enter digits
  - `+`, `-`, `*`, `/` – Choose an operation
  - `Enter` – Calculate the result (equals)
  - `Esc` – Clear the current entry
  - `Backspace` – Delete the last digit entered

---

The calculator layout adapts to various screen sizes. It looks great on desktop monitors and scales gracefully on smartphones and tablets, ensuring a smooth user experience across devices.
